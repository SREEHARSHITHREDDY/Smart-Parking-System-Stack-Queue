# Smart Parking System (Stack & Queue Based)

## Overview
This project simulates a Smart Parking System using Stack and Queue data structures in Python.

## Features
- Vehicle Parking (Stack)
- Waiting Queue (Queue)
- Remove Specific Vehicle
- JSON-based Data Persistence
- Menu-driven simulation

## Technologies Used
- Python
- OOP
- Stack & Queue
- JSON File Handling

## How to Run
python main.py
